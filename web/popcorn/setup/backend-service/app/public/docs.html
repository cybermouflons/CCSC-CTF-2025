API Documentation

Send you API key inside x-api-key HTTP header.

x-api-key: <your_api_key>

These are the available API endpoints for interacting with the Movie Review application.

POST

/api/login

Authenticate a user and return user details.

Example Request Body 
{
  "username": "admin",
  "password": "secret"
}
Example Response 
{
  "id": 1,
  "username": "admin",
  "password": ""
}
POST

/api/register

Register a new user.

Example Request Body 
{
  "username": "newuser",
  "password": "newpassword"
}
Example Response 
{
  "message": "User registered successfully!",
  "success": true
}
GET

/api/movies

Returns a list of available movies (id, title, image).

Example Response 
{
  "movies": [
    { "id": 1, "title": "Inception", "image": "/static/images/inception.jpg" }
  ]
}
GET

/api/movies/:id

Get full movie data and reviews.

Example Response 
{
  "movie": {
    "id": 1,
    "title": "Inception",
    "description": "...",
    "trailer": "...",
    "image": "..."
  },
  "reviews": [
    { "username": "user1", "review": "Great movie!" }
  ]
}
POST

/api/movies/:id/review

Submit a new review for a movie.

Example Request Body 
{
  "review": "Loved it!",
  "userID": 3
}
Example Response 
{
  "message": "Review added"
}
POST

/api/movies

Add a new movie entry.

Example Request Body 
{
  "title": "New Movie",
  "image": "https://example.com/image.jpg",
  "trailer": "https://youtube.com/embed/xyz",
  "description": "A brand new movie."
}
Example Response 
{
  "message": "Movie added"
}
GET

/users/:id/reviews

Get all reviews of a user.

Example Response 
{
  "reviews":[
    {
      "review":"Amazing soundtrack and storytelling.",
      "movie_title":"Interstellar"
    },
    {
      "review":"Heath Ledger stole the show!",
      "movie_title":"The Dark Knight"
    }
  ]
}
